# Claude Code for Beginners

Official sample code for **Claude Code for Beginners: Build Real Apps Step by Step -- No Coding Experience Required**.

This repository contains all project files from the book so you can follow along, compare your work, or use them as a starting point for your own projects.

## Projects

| Chapter | Project | What You Build |
|---------|---------|---------------|
| 3 | [HTML Game](ch03-html-game/) | Your first web page -- a simple browser game using HTML |
| 5 | [CSV Analyzer](ch05-csv-analyzer/) | A Python tool that reads and analyzes CSV data files |
| 8 | [Skills and MCP](ch08-skills-and-mcp/) | Configuration examples for Skills, Sub-agents, and MCP |
| 9 | [Quiz App](ch09-quiz-app/) | An interactive quiz app built with React |
| 10 | [Weather App](ch10-weather-app/) | A weather dashboard using React and a public API |

## Prerequisites

- A [Claude Pro plan](https://claude.ai/pricing) (or higher) for Claude Code access
- Python 3.10+ (Chapter 5)
- Node.js 18+ and npm (Chapters 9-10)

## How to Use This Repo

Each project folder contains the completed code from that chapter. If you get stuck while following the book, compare your files with the ones here.

You do **not** need to clone this repository to follow the book. Everything is built from scratch, step by step.

```bash
# If you want to browse locally:
git clone https://github.com/forest6511/claude-code-for-beginners.git
cd claude-code-for-beginners
```
