# Quiz App Project

React quiz application built with Vite.

## Project context

- Framework: React 18 + Vite
- Questions: 10 general knowledge questions in `src/App.jsx`
- Features: answer feedback (green/red), progress bar, results screen with score message
- Built in Chapter 9 of *Claude Code for Beginners*

## How to run

```bash
npm install      # first time only
npm run dev      # start development server at http://localhost:5173
```

## Project structure

- `src/App.jsx` — quiz logic and questions array
- `src/App.css` — styling
- `src/main.jsx` — React entry point

## When working with Claude Code

- To change questions: ask Claude Code to replace the questions array in App.jsx
- To add features: describe the feature (e.g., "add a countdown timer per question")
- To change styling: describe the look you want
